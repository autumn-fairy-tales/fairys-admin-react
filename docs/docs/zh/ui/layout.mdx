# 布局

整个项目的布局组件，包含侧边栏、顶部导航栏、内容区域等。

**引入**

```ts
import { Layout } from '@fairys/admin-tools-react';
```

## 头部信息

- 显示菜单,在布局设置`主菜单顶部+移入显示子菜单`、`主菜单顶部+子菜单侧边`时,会显示菜单。
- logo信息,在布局设置`主菜单顶部+移入显示子菜单`、`主菜单顶部+子菜单侧边`、`主子菜单合并侧边展示+顶部信息`时,会显示logo信息。
- 头像信息,在布局设置`主菜单顶部+移入显示子菜单`、`主菜单顶部+子菜单侧边`、`主子菜单合并侧边展示+顶部信息`、`主子菜单合并侧边展示+顶部信息`时,会显示头像信息。

## 侧边栏

- 显示主子菜单,在布局设置`主子菜单侧边`、`主菜单左侧+移入显示子菜单`、`主菜单顶部+移入显示子菜单`、`主菜单顶部+子菜单侧边`、`主子菜单合并侧边展示`、`主子菜单合并侧边展示+顶部信息`时,会显示菜单。

## 工具栏

- 显示面包屑
- 菜单搜索
- 面包屑
- 通知
- 全屏切换
- 刷新
- 切换主题

- 显示tab菜单
- 右键操作
