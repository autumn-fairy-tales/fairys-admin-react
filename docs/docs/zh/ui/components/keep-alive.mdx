# 缓存组件

缓存页面，在`react-activation`包的`AliveScope`的子组件中使用 KeepAliveBaseHOC 包裹组件，即可实现缓存功能。

:::tip 

这个组件当前在插件`路由生成`中使用。

:::

## 引入

```ts
import { KeepAliveBaseHOC } from '@fairys/admin-tools-react';
```

## 组件示例

```tsx
import { FairysFullScreen ,FairysButtonBase } from '@fairys/admin-tools-react';
import React,{ useState } from 'react';
import { FairysButtonBase } from '@fairys/admin-tools-react';

const Demo = () => {
  const [open,setOpen] = useState(false)
  return (<div style={{ position: "relative" , width: "100%", height: "300px" }}>
    <FairysButtonBase onClick={() => setOpen(true)}>
      打开全屏
    </FairysButtonBase>
    <FairysFullScreen open={open} onClose={() => setOpen(false)}>
      内容
    </FairysFullScreen>
  </div>);
};

export default KeepAliveBaseHOC(Demo,'keep-alive-demo');

```

## 组件参数

```ts
interface FairysFullScreenProps extends React.DetailedHTMLProps<React.HTMLAttributes<HTMLDivElement>, HTMLDivElement> {
  /**是否全屏*/
  open?: boolean;
  /**全屏内容*/
  children?: React.ReactNode;
  /**关闭全屏回调*/
  onClose?: () => void;
}
```
