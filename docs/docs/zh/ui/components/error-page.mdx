# 错误页面



## 引入

```ts
import { ErrorPage403, ErrorPage404, ErrorPage500 } from '@fairys/admin-tools-react';
```

## 组件示例

```tsx preview title="错误页面403"
import { ErrorPage403 } from '@fairys/admin-tools-react';

const Demo = () => {

  return (
   <div style={{ height: '500px' }} >
      <ErrorPage403 />
   </div>
  );
};
export default Demo;
```
