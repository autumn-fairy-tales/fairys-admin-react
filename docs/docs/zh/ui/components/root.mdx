# 根组件

FairysRoot 是 Fairys 框架的根组件，用于挂载页面缓存实例和处理路由导航。

## 引入

```ts
import { FairysRoot ,useFairysRootContext ,useFairysRoot, FairysRootContext} from '@fairys/admin-tools-react';
```

## 组件示例

```tsx
import { Fragment, useMemo } from 'react';
import { FairysRoot ,routerDataInstance} from '@fairys/admin-tools-react';

export const Login = () => {
  // 生成路由
  useMemo(() =>  routerDataInstance.createHashRouter([]), []);

  return <FairysRoot isOutletKeepAlive={false} router={routerDataInstance.router} keepAlive={true} />;
};
export default Login;
```

## 组件参数

```ts
import { DataRouter } from 'react-router';

export interface FairysRootProps {
  children?: React.ReactNode;
  /**启用缓存*/
  keepAlive?: boolean;
  /**路由*/
  router?: DataRouter;
  /**是否是Outlet组件启用缓存*/
  isOutletKeepAlive?: boolean;
}

```
